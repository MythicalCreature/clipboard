在它的配置文件里（通常是 ~/.continue/config.json）添加一个模型配置：

json
{
  "models": [
    {
      "title": "Spark Lite",
      "provider": "openai",
      "model": "lite",
      "apiBase": "https://spark-api-open.xf-yun.com/v1",
      "apiKey": "ETqUhynPAOZpFxmMfBSt:xWWbCpmluONOgYpHwygn"
    }
  ]
}
保存配置后，在 Continue 的对话界面里选择 Spark Lite 模型，就可以开始对话了。
